## 🖼️ 替换为真实温榆河全景图的方法

### 方案1：使用手机拍摄（最简单）

#### 推荐App：
1. **Google Street View App**（免费）
   - iOS: https://apps.apple.com/app/id904418768
   - Android: https://play.google.com/store/apps/details?id=com.google.android.street
   - 功能：自动拼接360°全景，可导出图片

2. **Cardboard Camera**（免费）
   - 简单易用的VR全景拍摄工具

#### 拍摄步骤：
1. 在温榆河绿道选择5个代表性位置
2. 打开App，选择拍摄全景
3. 按提示旋转手机360°
4. 保存并导出图片
5. 将图片放入项目的 `public/panorama/wenyu/` 目录

### 方案2：下载免费全景图（5分钟搞定）

#### 推荐网站：

**1. Pixabay 360°（完全免费，无需注册）**
```
https://pixabay.com/zh/images/search/360%20panorama/
```
- 搜索关键词：`nature 360`, `park 360`, `river 360`
- 所有图片免费可商用

**2. Flickr Equirectangular（大量资源）**
```
https://www.flickr.com/groups/equirectangular/
```
- 搜索：`river`, `park`, `greenway`
- 注意选择 CC0 或 CC BY 许可的图片

**3. Poly Haven（超高质量）**
```
https://polyhaven.com/hdris
```
- 提供8K高清全景图
- 完全免费，无需署名

#### 下载步骤：
1. 访问上述网站
2. 搜索相关关键词
3. 下载等距柱状投影格式的图片（Equirectangular）
4. 重命名为：`point1.jpg`, `point2.jpg`, 等
5. 放入 `public/panorama/wenyu/` 目录

## 🎯 图片要求

### 必须满足：
- ✅ **格式**: JPG 或 PNG
- ✅ **投影方式**: 等距柱状投影（Equirectangular）
- ✅ **宽高比**: 2:1（例如：4096x2048 或 8192x4096）

### 推荐参数：
- **分辨率**: 4096 x 2048 像素（高清）
- **文件大小**: 2-5 MB（压缩后）
- **格式**: JPG（质量85%）

### 如何检查图片是否可用：
1. 用图片查看器打开
2. 看起来像展开的地球仪（上下是极点，中间是赤道）
3. 宽度是高度的2倍
